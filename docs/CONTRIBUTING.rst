
Contributing
------------

TLDR
~~~~

* Use a virtualenv
* Use current versions of setuptools_ and zc.buildout_
* build and test
* Apply your changes; add new tests if necessary.
* Test again.
* Make a pull request.

.. _setuptools: https://pypi.org/project/setuptools
.. _zc.buildout: https://pypi.org/project/zc.buildout
